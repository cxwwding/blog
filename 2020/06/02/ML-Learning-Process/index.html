      • 机器学习学习进度

      Post on 2020-06-02

      前言

      最近在看《软技能-代码之外的生存指南》[John Z.Sonmez]一书,书中提到了要掌握一门技术，需要了解一下三个点：

      • 如何开始——要使用自己所学，需要掌握哪些基本知识？

      • 学科范围——我现在所学的东西范围有多大？我该怎么做？

      • 基础知识——了解基本的用户案例和最常见的问题

      下文我将根据上面提到的三个点回顾总结我的学习目标、目前的学习成果以及未来的学习路线。

      学习目标

      了解机器学习解决问题的基本思路，可以用Python、Matlab实现常用的算法。在学习结束后可以利用这些算法完成一个解决实际问题的练习。

      目前的学习成果

      • 用Excel、Matlab实现单变量、多变量的线性回归

      • 梯度下降算法在线性回归中的应用

      • 梯度下降算法在神经网络中的应用

      • BP算法的公式推导及原理

      • 使用Matlab的nnstart库来解决简单的神经网络问题

      未来的学习路线

      根据之前的学习，发现了存在一些问题：

      • 跟着视频学习的效率比较低下，视频中知识的转换率较低。

        解决方法：根据视频的标题，找到对应书本的章节自行学习，通过上网搜索资料解决问题。再选择性的观看自己不理解部分的视频。

      • 学过的内容遗忘速度比较快。

        解决方法： 在完整的学习一块内容后，用笔总结归纳要点。

      • 由于Python是初学的语言，对其的库的用法很不熟练也应为此更懒于使用Python编写程序。

        解决方法：将Python编程的任务，具体的规划出来。每一周分配固定工作量，周末总结。
